Today

本意只是想看看node.js，却用这个搭建了个独立博客，果然是个圈。

  1. Windows ENV
  2. download node.js
  3. download git bash
  4. get an gitHub account

steps at gitHub

  1. create a git repo:仓库名必须和git account name一致
    eg: https://github.com/yourname/yourname.github.io
  2. go to above repo setting, select master branch and save under “Source”

  3. Get you publishied pages
    eg: https://yourname.github.io/

    steps at localhost

  4. open cmd dialog
  5. install Hexo: npm install -g hexo
  6. select a directory as your project workplace
  7. cd into above directory
  8. 初始化：hexo init
  9. 生成静态页面: hexo g
  10. 启动本地服务: hexo s

  11. 访问 http://localhost:4000

    steps 关联本地和gitHub

  12. find “_config.yml” your local project workplace directory
  13. update “_config.yml” with following:
    deploy:
    type: git
    repo: https://github.com/yourname/yourname.github.io
    branch: master
  14. install hexo-deployer-git –save
  15. hexo d

  16. 访问 http://yourname.github.io/

    补充步骤：每次部署的步骤，可按以下命令来进行。
    hexo clean
    hexo g -d
